## 🛠️ Ferramentas e Tecnologias Utilizadas
- [Vs Code](https://code.visualstudio.com/)
- [Java](https://www.java.com/pt_BR/)

## 🌌 Funções
1. Inserir
2. Busca
3. Profundidade
4. Altura
5. Remove
6. Imprimir nos seguintes métodos: Prefixado, Interfixado, Posfixado
